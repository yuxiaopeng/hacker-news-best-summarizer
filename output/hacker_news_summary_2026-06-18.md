## 2. 洛尔——专为可扩展性设计的开源版本控制系统

**原文标题**: Lore – Open source version control system designed for scalability

**原文链接**: [https://lore.org/](https://lore.org/)

Lore 是 Epic Games 的下一代开源版本控制系统 (VCS)，旨在为数据和团队提供前所未有的可扩展性。它针对结合了代码和大型二进制资产（例如游戏和娱乐）的项目进行了优化，同时服务于开发人员和艺术家。

主要功能包括轻松设置、按需扩展、通过共享可重用数据和按需下载实现快速高效的工作流。它提供用于实验的轻量级分支、可验证且防篡改的历史记录、直观的命令行界面 (CLI) 以及跨多种编程语言（C/C++、C#、Rust、Go、Python、JavaScript）的广泛 API 支持，以便进行自定义集成。

Lore 采用中心化、内容寻址的架构，利用 Merkle 树和不可变版本链来确保加密完整性。它针对二进制优先存储、通过大文件分块进行重复数据删除以及稀疏工作区的“按需填充”（即用户只下载必要文件）进行了优化。缓存服务支持的架构通过避免数据重复的轻量级分支进一步提高了大型团队和存储库的吞吐量。

Lore 在 MIT 许可下发布，完全开源，体现了 Epic 对协作开发和开放标准的信念。鼓励用户加入 Discord 社区、查阅文档并在 GitHub 上做出贡献，GitHub 上也提供了各种语言的 SDK。
