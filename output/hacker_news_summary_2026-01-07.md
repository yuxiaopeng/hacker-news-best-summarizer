**原文链接**: [https://www.theregister.com/2026/01/05/aws_price_increase/](https://www.theregister.com/2026/01/05/aws_price_increase/)

AWS 在2026年1月的某个周六悄然上调了其面向机器学习的 EC2 容量块（特别是采用 NVIDIA H200 驱动的 p5e 和 p5en 实例）的价格，涨幅约为15%。例如，在大多数区域，p5e.48xlarge 实例的价格从每小时34.61美元跃升至39.80美元。AWS 将此次涨价归因于“供需模式”。

此次涨价意义重大，因为 AWS 历来对于核心计算服务倾向于降价或调整定价模型，而非直接、纯粹的提价。容量块对于那些需要有保障的 GPU 容量以运行高要求机器学习工作负载的公司至关重要，这些公司通常拥有数百万美元的预算。

作者 Corey Quinn 认为这是一个“风向标”时刻，表明这并非异常现象，而是新趋势的开端。他主张这打破了 AWS 二十年来让客户习惯于价格只会下降的惯性。这一先例使得未来其他受限资源（如内存或 Graviton 实例）的价格上涨更有可能。此举也为 Azure 和 GCP 等竞争对手提供了一个重要的谈资，并通过增加绝对成本影响了拥有企业折扣计划的客户。Quinn 总结说，云服务价格将永久下降的假设“在某个1月的周六消亡了”，标志着 AWS 定价策略在受到全球供应限制和成本上涨驱动下的根本性转变。

## 6. 后美国互联网

**原文标题**: The Post-American Internet

**原文链接**: [https://pluralistic.net/2026/01/01/39c3/](https://pluralistic.net/2026/01/01/39c3/)

科里·多克托罗在混沌通信大会上发表的演讲《后美国互联网》指出，唐纳德·特朗普的贸易政策无意中为一种新的、“抗粪化”的互联网创造了机会。作为电子前沿基金会（EFF）25年的活动家，多克托罗详细阐述了他对抗“反规避法”的斗争，这些法律将未经制造商许可修改数字产品的行为定为犯罪。

这些法律，例如美国《数字千年版权法》第1201条和欧盟《版权指令》第6条，源于美国，旨在惠及其科技巨头，使其能够控制用户体验并榨取价值（例如，苹果应用商店的佣金，约翰迪尔公司专有的维修系统）。至关重要的是，美国通过自由贸易协定，借由对关键出口产品征收关税的威胁，将这些法律强加给全球贸易伙伴。

多克托罗解释说，特朗普的单边关税尽管具有破坏性，但却消除了美国的强制性影响力。由于关税已成为现实，其他国家不再需要为了避免关税而维护反规避法。这种“解放日快乐”的局面允许各国作为一项战略回应，废除这些限制性法律，而不是简单地以自己的关税进行报复或屈服。

废除反规避法将赋能国内科技产业，使其能够创造“反粪化”产品——即允许用户重新掌控其设备和服务的解决方案。这将捕获目前流向美国科技巨头的经济价值。多克托罗设想了一个由数字权利活动家、关注数字主权的国家安全鹰派以及渴望竞争的本地企业组成的强大联盟，形成一股“不可阻挡”的力量，以建立这个后美国互联网。

## 7. Stop Doom Scrolling, Start Doom Coding: Build via the terminal from your phone

**原文标题**: Stop Doom Scrolling, Start Doom Coding: Build via the terminal from your phone

**原文链接**: [https://github.com/rberg27/doom-coding](https://github.com/rberg27/doom-coding)

This guide introduces "Doom Coding," a DIY method to code via your smartphone's terminal, anywhere with internet access, turning "doom scrolling" into productive building. Inspired by personal need, this setup allows users to access their main computer remotely.

To get started, you'll need a computer running 24/7, a smartphone, and a Claude Pro subscription, utilizing Tailscale, Termius, and Claude Code.

The setup involves three main steps:
1.  **Computer Setup:** Disable sleep, enable SSH/Remote Login, install and sign into Tailscale, and install Claude Code.
2.  **Phone Setup:** Install Tailscale and Termius (a mobile SSH client). Note your computer's MagicDNS address from Tailscale and create a new host entry in Termius with your computer's login credentials.
3.  **Connect and Code:** Activate the Tailscale VPN on your phone, select your configured host in Termius, and run `claude` to begin coding.

Troubleshooting tips include verifying your Tailscale VPN connection and ensuring your computer is on and unlocked. Best practices are shared for tracking progress (updating `CLAUDE.md`), previewing websites via HTTP servers, viewing databases with dedicated apps, and bookmarking useful development sites on your phone.

The author encourages contributions and feedback to enhance the "Doom Coding" experience.
