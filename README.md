指南中包含一套**热身练习**，其中有唱前检查清单（补水、姿势、释放张力、避免冷启动）以及一个5分钟的常规练习，包括呼吸练习、唇颤音、哼鸣、元音滑动和警报音练习。进阶热身则增加了吸管发声、舌颤音、琶音和轻柔的高音准备。

## 5. 展示 HN: 失忆后，如何重新访问电脑？

**原文标题**: Show HN: If you lose your memory, how to regain access to your computer?

**原文链接**: [https://eljojo.github.io/rememory/](https://eljojo.github.io/rememory/)

ReMemory is an open-source, browser-based tool designed to help users regain access to encrypted files if they become incapacitated or lose their memory. It encrypts files and splits the decryption key into multiple shares using Shamir's Secret Sharing. These shares are distributed among trusted friends, requiring a predetermined number (e.g., any 3 out of 5) to cooperate and reconstruct the key.

A core feature is that no single friend can access the data alone, enhancing security. Each friend receives a self-contained "recover.html" bundle that works offline, ensuring recovery is possible even if the original website vanishes or there's no internet. The entire process runs locally in the user's browser, meaning files never leave the device, ensuring privacy.

ReMemory is not a service, company, or cloud product, nor does it store user data. It's open-source (Apache-2.0), uses the 'age' encryption tool, and includes a self-audit document for transparency. Users can download demo bundles to experience the recovery process. The creator built ReMemory to offer a simple, secure, and decentralized way for friends to help in an emergency without relying on a single point of trust or third-party service.

## 8. SectorC: 仅512字节的C编译器 (2023)

**原文标题**: SectorC: A C Compiler in 512 bytes (2023)

**原文链接**: [https://xorvoid.com/sectorc.html](https://xorvoid.com/sectorc.html)

SectorC 是一款用 x86-16 汇编语言编写的突破性 C 编译器，它完全适配 512 字节的引导扇区，这使其可能成为有史以来最小的 C 编译器。它支持 C 语言的一个功能子集，包括全局变量、函数、`if`/`while` 语句、多种运算符、指针解引用和内联机器码。

该项目克服了在如此严苛的尺寸限制下进行词法分析和语法分析的最初挑战。“重大洞察 #1”是创建了“Barely C”——一种通过策略性空格分隔简化了标记，有效实现了“巨型标记”的变体。“重大洞察 #2”利用 `atoi()` 作为标识符和关键字的粗略哈希函数，消除了对复杂符号表和传统字符串到整数转换的需求。

在尝试 Forth 风格的“字节线程代码”被证明效率低下后，直接的递归下降解析器经过了精心的优化，从 468 字节缩小到 303 字节。这节省的 165 字节使得集成更多高级 C 语言特性成为可能。最小化技巧包括为实现贯穿、尾调用、调用融合而进行的大量代码重组，以及利用 `stosw`/`lodsw` 指令。一个 `binary_oper_tbl` 有效地添加了许多运算符。

为了节省空间，错误处理功能被有意省略。该编译器还与一个小型运行时（`rt/lib.c`, `rt/_start.c`）集成。示例展示了其执行低级任务的能力，例如在 VGA 模式 0x13 中动画正弦波，并通过 PC 扬声器播放音乐，这证明了在极端尺寸限制下，仍能实现显著的功能。
