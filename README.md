OpenAI争议性地从其使命宣言中删除了“安全地”一词，这预示着其优先事项已从安全转向利润，发生了重大转变。其2023年的使命旨在“构建能够安全地造福人类的人工智能，不受制于创造经济回报的需求”。然而，其2024年提交给美国国税局的文件显示，使命已更新为“确保通用人工智能造福全人类”，删去了“安全地”以及财务独立条款。

这一变化恰逢OpenAI从一个非营利组织转型为一个日益关注利润的商业实体，这主要是由人工智能开发所需巨额资本所驱动的。2025年10月，它正式重组为非营利性的OpenAI基金会和营利性的OpenAI集团。基金会目前仅持有集团26%的股份，实际上已放弃了多数控制权，而微软等主要投资者则持有大量股份。此举吸引了数十亿美元的新投资，使OpenAI的估值超过5000亿美元。

作者是一位非营利组织问责制学者，他认为这是一个关键但未被充分报道的转变，尤其考虑到针对OpenAI的诉讼悬而未决，这些诉讼指控其涉及心理操控和不当致死。尽管新结构包含一些安全委员会，但从核心使命中移除“安全地”一词，使得问责变得困难。作者批评OpenAI董事会以及加利福尼亚州和特拉华州的检察长允许这种情况发生，并建议采用其他模式，即非营利组织对其营利性对应实体保留更大的控制权，这将更好地服务于公共利益。

**原文标题**: My smart sleep mask broadcasts users' brainwaves to an open MQTT broker

**原文链接**: [https://aimilios.bearblog.dev/reverse-engineering-sleep-mask/](https://aimilios.bearblog.dev/reverse-engineering-sleep-mask/)

作者在官方应用出现问题后，对一款购自 Kickstarter 的智能睡眠眼罩进行了逆向工程。使用 Claude (Opus 4.6)，他们最初尝试进行蓝牙协议映射，但发现该协议并非标准。Claude 随后反编译了这款安卓 Flutter 应用的 APK，并分析了编译后的 Dart 二进制文件中的字符串。这揭示了关键信息：该公司 MQTT 消息代理的硬编码凭据、云 API 端点、命令函数名称以及数据包结构。借助 `blutter` 进一步分析，Claude 能够完全映射所有十五个设备命令，从而可以通过一个自定义的网页仪表板控制振动、加热和电肌肉刺激 (EMS) 等功能。
