这个未经专门数学训练的通用推理模型，提出了一个证明，构造了*n*个点的配置，其具有至少*n*^(1+δ)个单位距离点对，其中某个固定指数δ > 0 (例如，δ=0.014)，从而推翻了长期以来关于*n*^(1+o(1))的猜想。

该证明得到了蒂姆·高瓦斯和诺加·阿隆等著名数学家的验证和赞扬，称之为“AI数学领域的里程碑”。令人惊讶的是，该解决方案利用了代数数论中的高级概念，例如无限类域塔和戈洛德-沙法列维奇理论，在基本的几何问题之间建立了意想不到的联系。

这一成就标志着人工智能首次自主解决了某个数学子领域的核心重大开放问题，展示了深刻的推理能力。它预示着研究中人机协作的新时代，揭示了新的数学联系，并展示了人工智能作为跨科学和工程领域的强大研究伙伴的潜力。

## 5. 弗利珀一号——我们需要你的帮助

**原文标题**: Flipper One – we need your help

**原文链接**: [https://blog.flipper.net/flipper-one-we-need-your-help/](https://blog.flipper.net/flipper-one-we-need-your-help/)

Flipper One 是一个雄心勃勃的新项目，与 Flipper Zero 不同，旨在将 Linux cyberdeck 重新定义为一个开放、高性能的平台。它的创造者将其描述为一个“极其艰难的项目”，无论是在财务还是技术层面，因此正在寻求社区帮助。

Flipper One 旨在支持多种多样的用例，从 5G 网络分析到结合本地 AI 的 SDR，并通过 M.2 模块（支持 PCI Express、USB 3.0、SATA，用于调制解调器、SDR、SSD、AI 加速器）和 GPIO 连接器实现广泛的硬件扩展。它提供强大的网络连接能力，包括两个千兆以太网口、USB 以太网和 Wi-Fi 6E。

一个核心目标是构建最开放的 ARM 计算机，确保瑞芯微 RK3576 SoC 获得完整的上游 Linux 内核支持，且不含二进制私有驱动。他们已与 Collabora 合作以实现这一目标，并正在征集社区帮助以消除最后一个二进制私有驱动（DDR 训练器）并将剩余驱动程序上游化。

该设备采用独特的协处理器架构，将一个 8 核 RK3576 CPU（运行 Linux）与一个低功耗 RP2350 微控制器（管理显示、电源和输入）配对，即使 CPU 关闭也能实现基本功能。

为了增强移动 Linux 体验，Flipper One 推出了 Flipper OS，这是一个基于 Debian 的系统，带有用于管理不同软件配置的“配置文件”。他们还在开发 FlipCTL，一个用于小屏幕的 UI 框架，旨在为现有 Linux 实用程序创建直观的、基于菜单的界面，类似于 Flipper Zero 备受赞誉的用户界面。

“Flipper One 开发者门户”提供了一个开放的开发流程，共享硬件、Linux 软件、用户界面等各个子项目的内部文档、任务追踪器和讨论，邀请所有爱好者贡献力量。
