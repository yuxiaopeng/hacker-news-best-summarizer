欧洲央行行长克里斯蒂娜·拉加德（Christine Lagarde）敦促欧洲建立自己的数字支付系统，她警告称，目前对Visa、万事达卡（Mastercard）、PayPal和支付宝等支付基础设施的依赖，会导致欧洲消费者数据被发送到美国或中国。这种每年处理24万亿美元交易的依赖，造成了重大的地缘政治脆弱性，正如2022年俄罗斯被切断与这些网络的连接时所造成的混乱所凸显的那样。

作为一名英国公民，Thomas-Johnson 认为 ICE 试图追踪并拘留他，但他已逃往塞内加尔。尽管经历了“可怕”的经历，他仍致力于新闻事业。谷歌未回应置评请求。

**Discord (F:4, O:1, Se:3, Sa:4, D:1)** serves its purpose well for real-time chat and moderation but lacks openness, true E2EE for text, and decentralization. Historical content is hard to navigate.

**Signal (F:2, O:4, Se:5, Sa:2, D:1)** is the gold standard for secure, E2EE communications, ideal for secrecy-focused communities. However, it severely lacks community-building features like threading, organized channels, pinning, and granular moderation, sacrificing these for privacy.

**Matrix (F:3, O:4, Se:3, Sa:1, D:4)** offers a federated, encrypted protocol with a Discord-like client (Element). While promising decentralization, its complexity, poor native moderation tools (leading to spam issues), and heavy reliance on the central matrix.org server undermine its practical decentralization and require significant admin effort.

**Rocket.Chat (F:5, O:3, Se:4, Sa:3, D:3)** provides a self-hostable, Slack-like experience with high functionality and E2EE. It operates on a freemium model, limiting features for unpaid users, but offers independence for those willing to pay. Administrators can audit messages, which is part of its documented moderation.
