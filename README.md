不列颠哥伦比亚省省长戴维·埃比宣布，该省将永久实行夏令时，2026年3月8日将是大多数居民最后一次将时钟“往前拨快一小时”。从2026年11月1日起，当钟表通常“往回拨慢一小时”时，不列颠哥伦比亚省将全年保持夏令时，并将其新时区命名为“太平洋时间”。不列颠哥伦比亚省东部一些沿用山区时间的社区将不受影响。

埃比通过指出时制转换带来的问题，例如睡眠不足、车祸增加以及宠物生活规律被打乱，来解释此决定的原因。一份2019年的公众参与报告显示，93%的受访者支持全年实行夏令时，主要出于健康和福祉的考虑。

尽管不列颠哥伦比亚省于2019年通过了相关立法，但此前一直在等待华盛顿州、俄勒冈州和加利福尼亚州等美国邻邦采取行动。然而，由于美国立法仍悬而未决，不列颠哥伦比亚省在育空地区类似改革的启发下，正在单方面采取行动。不列颠哥伦比亚大学教授维尔纳·安特维勒支持这一举措，强调了时制转换带来的“时差感”，并认为不列颠哥伦比亚省可能会促使其他地区效仿。

然而，这一决定引发了褒贬不一的反应。商界领袖，包括大温哥华贸易局和温哥华机场管理局，表达了担忧，认为单方面行动将给跨境企业带来“麻烦”，并扰乱航空运营，强调与邻近司法管辖区保持一致的重要性。加拿大独立企业联合会则认为，此举可能是为了转移人们对省级赤字的注意力。
