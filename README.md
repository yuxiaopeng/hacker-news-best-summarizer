## 8. The Illustrated Transformer

**原文标题**: The Illustrated Transformer

**原文链接**: [https://jalammar.github.io/illustrated-transformer/](https://jalammar.github.io/illustrated-transformer/)

The Transformer, introduced in the "Attention is All You Need" paper, is a deep learning model designed to improve machine translation speed and training parallelization. It consists of an encoder-decoder structure, with each component being a stack of identical sub-layers.

Each encoder processes input sequences through a self-attention layer, which helps it weigh the importance of other words when encoding a specific word, followed by a position-wise feed-forward neural network. Decoders similarly use self-attention and feed-forward networks, adding an extra attention layer to focus on relevant parts of the encoder's output.

Inputs begin as word embeddings. Crucially, to account for word order, positional encoding vectors are added to these embeddings. Self-attention works by creating three vectors for each word: Query (Q), Key (K), and Value (V). Attention scores are calculated by dot-producting Q with K, scaling, applying a softmax function, and then summing the weighted V vectors. This process allows the model to identify relevant contextual words.

Multi-headed attention enhances this by performing the self-attention calculation multiple times with different learned Q/K/V weight matrices. This enables the model to focus on various parts of the input sequence simultaneously and learn diverse contextual representations. The outputs from these heads are then combined. This parallelizable architecture makes the Transformer highly efficient, distinguishing it from traditional sequential models. The concepts have evolved and are now covered in an updated book.
