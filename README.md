## 1. 不要把儿童保护变成网络管制。

**原文标题**: Do Not Turn Child Protection into Internet Access Control

**原文链接**: [https://news.dyne.org/child-protection-is-not-access-control/](https://news.dyne.org/child-protection-is-not-access-control/)

本文反对将年龄验证从成人网站扩展到主流互联网服务，认为这会将互联网从一个开放网络转变为一个“许可访问”系统。尽管承认保护儿童免受网络危害的真实需求，作者坚称，年龄验证，特别是在集成到操作系统中时，会作为一个通用的身份层，混淆了内容审核与监护权的明确责任。

作者指出，这种转变削弱了所有人的隐私，集中了控制权，并使责任从父母、教师和社区手中转移。这被批评为一个昂贵、易于规避的“企业数据攫取”，其背后是那些从数据收集中获利的公司游说推动的。作者警告说，为年龄验证而构建的基础设施很容易被重新用于其他属性，例如位置或公民身份，从而为互联网访问设置一个普遍的门槛。

提出的解决方案包括本地内容审核（例如，在浏览器、设备、学校网络中进行）并将监护权保留给能够做出情境化判断的家庭和当地社区。操作系统应该作为用户控制下的本地策略界面，而不是一个普遍的年龄广播层。文章主张不进行一刀切的年龄检查，而是倡导监管有害的推荐系统、黑暗模式以及放大不负责任内容的商业模式。

作者Jaromil详细阐述了他反对这一趋势的历程，包括因年龄验证而停止为Dyne.org使用Discord，宣布Devuan GNU/Linux拒绝实施此类功能，并在卢加诺启动了一个本地管理监护的试点项目。他敦促大家参与进来，以确保互联网保持开放，同时通过相称的、本地化的解决方案有效保护儿童。

## 3. 移民欧盟

**原文标题**: Migrating to the EU

**原文链接**: [https://rz01.org/eu-migration/](https://rz01.org/eu-migration/)

作者正在系统地将各种在线服务和订阅从非欧盟国家的服务商迁移到欧盟的服务商。这一决定是出于当前的全球政治局势以及对更好数据保护的渴望，充分利用欧盟用户友好的法律。此次迁移旨在以可承受的价格提供相同或更好质量的服务。

主要迁移包括：
*   **邮件：** 从Fastmail迁移到Uberspace（德国），这是一家灵活的“随心付”服务商，提供电子邮件、网站和Shell托管服务。现在使用适用于Android的Thunderbird作为客户端。
*   **日历：** 由于Uberspace缺少内置日历功能，所以在Uberspace Asteroid上安装了NextCloud，以提供CalDAV和CardDAV支持。客户端包括Thunderbird、DAVx5和Fossil Calendar。
*   **网站托管：** 整合到Uberspace，取代了之前的Hetzner VPS。
*   **域名和DNS：** 从Namecheap切换到hosting.de（德国），原因在于其地理位置、价格和支持。
*   **Git：** 将代码库从GitHub和自托管的Gitea迁移到德国非营利组织Codeberg.org。
*   **VPN：** 继续使用Mullvad（瑞典），理由是其无日志政策、直接透明的定价和可靠性。
*   **安卓/谷歌生态系统：** 对于个人移动使用，采用了一部搭载Graphene OS的Pixel 9a，并整合了F-Droid和Google Play商店，以实现应用的灵活性（例如，使用NewPipe实现无广告YouTube）。一台Chromebook正在被一台运行MX Linux和Fluxbox的11英寸MacBook Air所取代。

## 6. Tinybox — 强大的深度学习计算机

**原文标题**: Tinybox – A powerful computer for deep learning

**原文链接**: [https://tinygrad.org/#tinybox](https://tinygrad.org/#tinybox)

tiny corp 推出了 **tinygrad**，这是一个简单而强大的神经网络框架，它将复杂操作分解为 ElementwiseOps（逐元素操作）、ReduceOps（归约操作）和 MovementOps（移动操作）三种类型。它旨在通过编译定制的、积极融合的内核并维护一个更简单的后端，从而超越 PyTorch 的速度。tinygrad 目前处于 alpha 阶段，但已被 openpilot 用于深度学习模型。

该公司还销售 **tinybox**，这是一系列强大的深度学习计算机。目前有售价 12,000 美元的 **red v2**，提供 778 TFLOPS FP16 性能和 64GB GPU 内存；以及售价 65,000 美元的 **green v2**，拥有 3086 TFLOPS FP16 性能和 384GB GPU 内存。未来还计划推出 **exabox**（2027 年，约 1000 万美元），目标是实现约 1 EXAFLOP 的性能。这些机器以其卓越的性价比而著称，并在 MLPerf Training 4.0 基准测试中得到了验证。

tinybox 订单仅通过电汇处理，不提供定制选项，以保持低价格和高质量；全球发货，一周内送达。tiny corp 正在积极招聘工程师、运营人员和硬件专家，其核心使命是通过将千万亿次浮点运算能力商品化并让每个人都能使用先进计算来加速 AI 的发展。

## 7. 游牧项目 — 永不离线的知识

**原文标题**: Project Nomad – Knowledge That Never Goes Offline

**原文链接**: [https://www.projectnomad.us](https://www.projectnomad.us)

NOMAD项目（离线媒体、档案和数据节点）是一个免费、开源的离线服务器解决方案，使用户无需互联网连接即可访问海量知识和工具。它可安装在任何个人电脑上，包含一个离线信息库（通过Kiwix访问维基百科、古腾堡计划）、一个本地AI助手（Ollama支持大型语言模型）、全面的离线地图（OpenStreetMap）和一个教育平台（通过Kolibri访问可汗学院课程）。

NOMAD主要为应急准备、离网生活、技术爱好者和教育环境而设计，确保即使在互联网基础设施故障或不可用时，关键信息和学习资源依然可访问。相较于PrepperDisk或Doom Box等竞争对手，NOMAD项目的一个主要优势是它完全免费、开源（Apache 2.0许可证），并在用户选择的硬件上运行，支持GPU加速的AI以提供更高级的智能。这与那些专有、昂贵且通常锁定在功能有限的树莓派硬件上的替代方案形成了鲜明对比。

安装过程简单，在Ubuntu或基于Debian的Linux系统上只需两条命令即可完成。NOMAD项目倡导数字独立性，并通过社区资助来保持其免费地位。用户可以通过GitHub访问并订阅更新。
