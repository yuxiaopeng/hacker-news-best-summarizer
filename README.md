要取消您的ChatGPT Plus订阅，请按照以下步骤操作：

1.  **登录ChatGPT：** 在您的网络浏览器中访问ChatGPT。
2.  **访问您的设置：** 在左下角，点击您的电子邮件地址或用户名。
3.  **管理我的订阅：** 从弹出的菜单中，选择“管理我的订阅”。这将把您重定向到您在Stripe（OpenAI的支付处理器）上的账户页面。
4.  **取消套餐：** 在Stripe门户网站上，找到并点击“取消套餐”按钮。
5.  **确认取消：** 按照屏幕上的任何提示确认您的取消决定。

取消后，您的订阅将在当前计费周期结束前保持有效。您不会在下一个计费周期被收费。OpenAI通常不为在计费周期中途取消的订阅提供按比例退款，但您将保留对ChatGPT Plus功能的访问权限，直到您当前的付费期结束。

## 10. A new California law says all operating systems need to have age verification

**原文标题**: A new California law says all operating systems need to have age verification

**原文链接**: [https://www.pcgamer.com/software/operating-systems/a-new-california-law-says-all-operating-systems-including-linux-need-to-have-some-form-of-age-verification-at-account-setup/](https://www.pcgamer.com/software/operating-systems/a-new-california-law-says-all-operating-systems-including-linux-need-to-have-some-form-of-age-verification-at-account-setup/)

A new California law, Assembly Bill No. 1043, will mandate age verification for all operating system providers, effective January 1, 2027. Approved by Governor Gavin Newsom, the law requires OS providers to implement an accessible interface at account setup to collect a user's birth date or age.

Additionally, OS providers must provide a real-time API signal to application developers upon request, categorizing users into four age brackets: under 13, 13-15, 16-17, and 18 and over. While the bill doesn't demand invasive methods like face scans, it necessitates the collection and segmentation of user age information.

This legislation might not significantly affect systems like Windows, which already gather birth dates for account setup. However, it has drawn considerable concern from Linux communities. Critics argue that compliance would be challenging or impossible for open-source operating systems, suggesting that Linux distributions might issue disclaimers for California users or that users would simply choose non-compliant versions.

The article places California's initiative within a broader global trend of governments pushing for mandatory age verification, citing examples like the UK's Online Safety Act. These efforts frequently face criticism regarding privacy concerns and data handling, underscoring the ongoing debate despite the practical difficulties of enforcing such laws, particularly for diverse operating systems.
