## 7. 电视今天一百岁了

**原文标题**: Television is 100 years old today

**原文链接**: [https://diamondgeezer.blogspot.com/2026/01/tv100.html](https://diamondgeezer.blogspot.com/2026/01/tv100.html)

今天，即2026年1月26日，是苏格兰发明家约翰·洛吉·贝尔德首次公开演示电视的100周年纪念日。尽管早期的实验在其他地方进行，尤其是在黑斯廷斯，贝尔德于1923年使用家居用品制造了他的第一个发射器并实现了第一张图像（圣约翰救护车奖章的阴影），但决定性时刻发生在伦敦。

1924年11月，贝尔德搬到苏荷区弗里斯街22号，在一家意大利咖啡馆楼上设立了工作室。在塞尔福里奇百货公司演示了他的机电式“电视机”（Televisor）后，他完善了自己的技术。到1925年10月，他成功传输了具有明暗渐变的图像，其中最著名的是使用了假人“Stooky Bill”，随后是20岁的威廉·泰恩顿，他成为第一个被电视直播的人。

1926年1月26日，在弗里斯街，贝尔德向新闻界和皇家研究院成员进行了官方演示，展示了假人，随后传输了人脸图像。尽管当时在场的许多人没有完全领会其重要性，《泰晤士报》仍对贝尔德的发明进行了报道。

贝尔德继续创新，到1928年开发出了彩色电视、立体（3D）电视和红外电视。他于1930年启动了BBC的实验性电视传输。然而，当广播电视于1936年推出时，他的机械系统与马可尼-EMI更优越的电子系统展开竞争，并在三个月后被淘汰。

尽管由于竞争和战时中断，贝尔德的光芒逐渐减弱，但他因使电视成为可能而被铭记。今天，弗里斯街22号，即现在的意大利酒吧（Bar Italia），被授予牌匾，作为电视直播的“世界起源地”，以纪念贝尔德的开创性贡献，正是他的贡献使电视成为20世纪一项具有变革意义的发明。
