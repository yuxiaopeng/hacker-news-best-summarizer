## 2. Cloudflare 收购 Astro

**原文标题**: Cloudflare acquires Astro

**原文链接**: [https://astro.build/blog/joining-cloudflare/](https://astro.build/blog/joining-cloudflare/)

Cloudflare 已收购 Astro 技术公司，该公司是广受欢迎的 Astro 网络框架的幕后公司。所有 Astro 全职员工现在将加入 Cloudflare，继续他们在该框架上的工作。

Astro 将保持开源、MIT 许可、积极维护，并继续支持广泛的部署目标，而不仅仅是 Cloudflare。其开放治理模式和当前路线图也将保持不变。

Astro 诞生于 2021 年，旨在构建高性能、内容驱动的网站，以对抗臃肿、以应用程序为中心的网络架构趋势。尽管获得了显著关注——每周近 100 万次下载并被主要公司采用——但围绕托管原语构建业务的尝试被证明不成功，并分散了核心框架开发的精力。

此次收购将 Cloudflare 的基础设施端愿景（全球、快速、安全的平台）与 Astro 的框架端愿景（易于构建、默认快速的网站）结合起来。作为长期以来的开源支持者，Cloudflare 将为 Astro 提供所需的资源，使其能够完全专注于创新和代码。该团队现在可以百分之百地投入到框架的增强中，Astro 6 等即将发布的版本旨在改进网络内容交付的性能、可扩展性和可靠性。

## 4. 东德气球逃生

**原文标题**: East Germany balloon escape

**原文链接**: [https://en.wikipedia.org/wiki/East_Germany_balloon_escape](https://en.wikipedia.org/wiki/East_Germany_balloon_escape)

1979年9月16日，来自东德的施特雷尔齐克和韦策尔两个家庭共八人，乘坐自制热气球大胆逃往西德。面对戒备森严的德国内部边境，彼得·施特雷尔齐克和京特·韦策尔于1978年3月受电视节目启发，开始策划这一行动。

这一精心策划的行动，准备工作持续了一年半多。他们费尽心力，从各地偏远的城镇采购大量布料，以避免引起怀疑。由于布料多孔或燃烧器动力不足等问题，许多原型气球和试飞都失败了。韦策尔曾几乎放弃，但施特雷尔齐克坚持不懈，并发现了一种更有效利用丙烷的方法。

1979年7月，施特雷尔齐克一家进行了首次单独逃亡尝试。他们的气球过早下降，距离边境仅180米，迫使他们徒步潜回，未被发现。然而，被遗弃的气球被东德当局（史塔西）发现，这使得两个家庭在被捕前再次尝试的紧迫性大大增加。

两个家庭迅速建造了第三个、更大的气球，直径达20米。1979年9月15日，所有八人一同升空。尽管起飞时发生小火，且飞行途中气球出现破裂，导致燃烧器反复熄灭，但韦策尔每次都设法将其重新点燃。经过28分钟的飞行，达到2500米的高度后，他们安全降落在西德巴伐利亚州的奈拉附近，只有韦策尔摔断了一条腿。

东德对此作出反应，加强了边境安全，并限制了适合建造气球的材料。施特雷尔齐克的兄弟姐妹因协助逃亡而被捕入狱，后来在国际特赦组织的帮助下获释。这两个家庭最初在奈拉定居，施特雷尔齐克一家最终在德国统一后回到了他们的家乡。

## 5. 当互联网中断时，Briar 通过蓝牙和 Wi-Fi 帮助伊朗保持连接。

**原文标题**: Briar keeps Iran connected via Bluetooth and Wi-Fi when the internet goes dark

**原文链接**: [https://briarproject.org/manual/fa/](https://briarproject.org/manual/fa/)

Briar是一款去中心化消息应用程序，专为活动家、记者以及任何需要安全和弹性通信的人设计。与传统通讯工具不同，它无需中央服务器，直接在设备间同步消息。

一个关键特性是它无需互联网即可运行，通过蓝牙或Wi-Fi直连在危机期间确保通信畅通。当互联网可用时，Briar利用Tor网络（包括“网桥”）保护用户隐私免受监视并绕过审查。

可在Android设备上使用，用户创建本地账户；没有云备份，这意味着忘记密码或删除应用程序会导致数据丢失。联系人可以通过共享链接远程添加，也可以通过扫描二维码安全地面对面添加。所有通信均采用端到端加密。

Briar提供多种通信功能：私人一对一消息、私人群组（创建者管理邀请）和公共论坛（任何成员都可以邀请他人）。它还包括一个个人博客功能，自动与联系人共享；以及一个RSS订阅阅读器，通过Tor私密访问新闻。

设置允许用户自定义主题、控制Tor连接方式（带或不带网桥）、管理移动数据使用，并启用屏幕锁定以增强应用程序安全性。Briar兼具离线本地连接和通过Tor实现在线隐私的双重能力，使其成为在充满挑战的环境中保持联系的强大工具。

## 6. 只是浏览器

**原文标题**: Just the Browser

**原文链接**: [https://justthebrowser.com/](https://justthebrowser.com/)

“纯粹浏览器”是一个开源项目，旨在通过移除AI功能、遥测数据、赞助内容、购物集成及其他冗余功能来精简流行的桌面网页浏览器。其目标是利用通常用于企业IT环境的隐藏组策略设置，在不修改浏览器可执行文件的情况下，提供极简的浏览体验。

该项目为Google Chrome、Microsoft Edge和Mozilla Firefox（不包括Linux上的Chrome/Edge）提供配置文件、文档和简易安装脚本。用户可以通过PowerShell（Windows系统）或终端（Mac/Linux系统）中的简单命令安装这些配置，或遵循手动指南进行操作。

主要移除的功能包括大部分生成式AI功能、价格追踪器、优惠券代码、推荐文章、默认浏览器提示、首次运行体验以及大部分遥测数据。如果崩溃报告是一个独立选项，通常会保持启用。尽管浏览器可能会因为这些策略显示“由组织管理”的信息，但这些设置是持久的，并且可以定制或移除。“纯粹浏览器”不包含广告拦截器，但推荐使用。它旨在改进主流浏览器，并承认替代品可能缺乏持续的安全更新或广泛的平台可用性。目前尚不支持移动设备。
