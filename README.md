## 8. 声波攻陷者：利用扬声器，无需接触即可入侵你的电脑

**原文标题**: Pwnd Blaster: Hacking your PC using your speaker without ever touching it

**原文链接**: [https://blog.nns.ee/2026/06/03/katana-badusb/](https://blog.nns.ee/2026/06/03/katana-badusb/)

一位安全研究员对创新声霸锣 Katana V2X 音箱的固件进行了逆向工程，发现了严重漏洞。该音箱通过 USB 使用自定义的“创新传输协议”（CTP），这需要身份验证。然而，CTP 也可以通过低功耗蓝牙 (BLE) 访问，无需任何身份验证或配对，允许附近任何攻击者（约15米范围内）发送命令。

进一步分析发现，固件更新（同样通过 CTP 执行）缺乏适当的签名检查，仅依赖一个微不足道的 SHA-256 校验和。这意味着攻击者可以制作并上传恶意的自定义固件，通过未经身份验证的蓝牙远程上传到音箱。

其影响是严重的：该音箱通常通过 USB 连接到电脑，并带有一个麦克风。攻击者可以上传固件，将其变成秘密监听设备，传输音频。更为关键的是，研究员演示了修改固件，使音箱模拟为一个完整的 USB 键盘。这允许远程攻击者上传固件，在重启后，向连接的电脑注入任意按键输入，执行命令（例如：“echo pwned”）。这有效地制造了一种“远程 Rubber Ducky”攻击，无需物理接触或配对。

创新公司的回应是，他们不认为这是一个网络安全风险，并拒绝发布补丁。最新的官方固件仍然存在漏洞。该研究员已发布一个工具，通过修补固件来阻止 CTP 通过蓝牙传输，从而提供一个部分补救措施，但这可能会禁用官方的移动应用程序。
